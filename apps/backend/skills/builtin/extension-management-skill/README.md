# 扩展管理指南

让 Agent 自主管理 AIASys 中的扩展：MCP Server、协作专家（Expert）和 Skill。

## 适用场景

- 搜索并安装外部 MCP Server 到工作区
- 搜索并安装外部 Skill 到工作区
- 安装系统内置协作专家到全局或工作区
- 配置工作区中已安装 MCP Server 的启用工具列表
- 查看当前已安装的 MCP、Expert、Skill 清单

## 核心能力

| 能力 | 说明 |
|------|------|
| MCP 市场搜索与导入 | 浏览外部 MCP 市场，导入到我的默认仓库，再复制到工作区 |
| MCP 工作区配置 | 查看生效配置、测试连接、管理启用的工具 |
| 专家安装 | 将系统内置专家安装到全局或当前工作区 |
| 专家策略 | 查看和修改工作区/全局的协作专家启用策略 |
| Skill 市场搜索与安装 | 浏览外部 Skill 市场，直接安装到工作区 |
| Skill 仓库管理 | 查看 Skill 仓库，启用/禁用/更新工作区 Skill |
