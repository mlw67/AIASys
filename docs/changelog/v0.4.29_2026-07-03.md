# Changelog v0.4.29 - 2026-07-03

**更新日期**: 2026-07-03

## 修复 (Bug Fixes)

### 飞书 Bad Case 修复（9 条）

- **BC-005: Provider 更新丢失类型**
  - `UpdateProviderRequest` 增加 `type` 字段，更新 provider 时不再清空接口格式类型。

- **BC-006: Windows `C:\workspace` 路径未重定向**
  - Agent Host 提示词注入当前工作区物理路径与 `C:\workspace` 重定向警告。
  - Shell 工具执行前将 `C:\workspace` 重写为真实工作区路径。

- **BC-007: Shell 环境回退与 PowerShell 转义不一致**
  - `ShellExecutor` 彻底禁用 `cmd.exe`，统一降级到 PowerShell。
  - `runtime_execution` 按目标 shell family 选择正确的命令转义方式。
  - 修复 Windows 上 `Path(PosixPath)` 导致的崩溃。

- **BC-008: 新建工作区模板选择位置靠后**
  - 新建工作区对话框中，模板选择块前移到运行环境配置之前。

- **BC-009: 子 Agent 流式输出被拆成多条消息**
  - 同一子 Agent 任务的多段 `subagent_content` 流式输出合并为单条 ChatItem。
  - 备注：此为过渡方案，完整对话迁到 tab 页的方案已认领为后续任务。

- **BC-011: 默认工作区路径/ID 不直观**
  - 默认存储路径改为 `~/AIASys/{data,logs,workspaces}`。
  - 工作区 ID 格式改为 `YYYYMMDD-HHMMSS-XXXX`。
  - 存储设置增加默认路径展示与“恢复默认”按钮。

- **BC-012: Web 端显示无效的“打开文件所在目录”菜单**
  - Web 端文件树右键仅在桌面 IPC 可用时显示“打开文件所在目录”等菜单项。

- **BC-014: 前端未展示版本号**
  - 桌面端新增 `getVersion()` / `openExternal()` IPC。
  - 设置页新增“关于 AIASys”区域，展示版本号与手动检查更新入口。

- **BC-017: 能力列表图标单一**
  - `CapabilityPanel` 能力列表按 `kind` 显示不同图标。

### 其他修复

- **Shell 解释器识别**
  - `ShellExecutor._detect_family_from_name()` 支持 `sh.exe` / `bash.exe` 等带 `.exe` 后缀的解释器，避免被误判为 `custom`。

## 回退与待决策

- **BC-013: thinking/reasoning 输出处理**
  - 已回退 v0.4.28 中新增的 `thinking_disabled` 字段、UI 开关和 `<think>` 标签剥离逻辑。
  - 原因：不同厂商实现差异大，直接剥离 `<think>` 标签可能误伤正常输出，需产品层面明确策略后再处理。
  - 详见本地 issue：`design-draft/bug-discovery/BC-013-thinking-rendering-strategy.md`。
