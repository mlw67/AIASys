# Changelog v0.4.30 - 2026-07-04

**更新日期**: 2026-07-04

## 修复 (Bug Fixes)

### 子 Agent tab 交互修复

- **修复子 Agent tab 内工具调用按钮点击无响应**
  - `SubagentTabContent` 为 `<ChatArea>` 传入 `onViewToolDetails` 回调。
  - 点击 `tool_call` 片段按钮时，通过 `resolveToolPreviewFromEvents` 解析工具参数与执行结果，弹出与主对话一致的 `ToolPreviewPopover` 悬浮窗。

- **修复子 Agent tab 切换时页面闪烁**
  - `PaneRenderer` 将 `tab.subagentId` 加入 keep-alive 列表。
  - 切换 tab 时子 Agent 内容保持挂载，避免重新加载详情导致的闪烁。

- **保留子 Agent 历史事件文本空白**
  - `subagent_tracking_service.py` 持久化 `text` 事件时保留原始空白，避免代码块等格式错乱。
